Batch signature of Signer documents with Web PKI

Select a certificate and click "Sign Batch". The documents will then be created on Lacuna Signer as public signatures for the selected certificate's holder and signed in batch: for each document the browser starts the signature (getting the hash from Signer), signs the hash with Web PKI and notifies Signer that the signature is complete.
